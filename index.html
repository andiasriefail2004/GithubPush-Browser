Belum punya token? Buat token baru di GitHub → Pilih scope repo saja, dan set masa berlaku pendek (7 hari) kalau cuma untuk coba-coba.
⚠️
Mode testing pribadi. Kolom Secret di bawah cuma aman kalau kamu satu-satunya yang buka halaman ini. Jangan pernah commit Secret asli ke repo GitHub — kolom ini sengaja tidak menyimpan Secret permanen, kamu isi ulang tiap kali pakai.
